# Diplom-3.0 - Система анализа теплопотребления МКД

## 📋 Описание

Система предназначена для сбора, обработки и анализа данных о теплопотреблении многоквартирных домов (МКД). Реализована в рамках магистерской диссертации.

### Ключевые возможности

- **Загрузка данных** из Excel файлов (.xlsx, .xls)
- **Предобработка**: валидация диапазонов, интерполяция пропусков, расчёт ГСОП
- **Статистический анализ**: линейная и робастная регрессия, квантильная регрессия
- **Выявление аномалий**: EWMA, Isolation Forest, тест Граббса
- **Кластеризация**: K-Means с оценкой силуэта
- **Прогнозирование**: Холт-Винтерс (тройное экспоненциальное сглаживание)
- **Оценка эффективности**: сравнение с нормативами, рекомендации
- **Веб-интерфейс**: интерактивные графики Plotly, экспорт отчётов

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

Или вручную:



```bash

```

```bash

python src/preprocess.py

python src/analytics.py
```

## 📊 Формат входных данных

Excel файл должен содержать следующие колонки:

| Колонка | Описание | Тип | Пример |
|---------|----------|-----|--------|
| mkd_id | Идентификатор МКД | str | MKD-001 |
| date | Дата | datetime | 2023-01-01 |
| T_out | Температура наружного воздуха | float | -5.2 |
| T_in | Температура внутри помещения | float | 22.5 |
| Q | Тепловая нагрузка | float | 1.85 |

## 🔧 Настройка

### Валидация диапазонов

В `src/loader.py` можно изменить допустимые диапазоны:

```python
VALID_RANGES = {
    'T_out': (-40, 40),      # Температура наружного воздуха
    'T_in': (18, 26),        # Температура внутри помещения
    'Q': (0, 1000),          # Тепловая нагрузка (Гкал/сутки)
}
```

### Расчёт ГСОП

В `src/preprocess.py` базовая температура (по СП 50.13330.2012):

```python
BASE_TEMP = 18.0  # °C
```

## 📈 Алгоритмы

### Регрессионный анализ

- **Линейная регрессия**: Q = α + β·T_out + ε
- **Робастная регрессия (Huber)**: устойчива к выбросам
- **Квантильная регрессия**: доверительные интервалы

### Выявление аномалий

- **EWMA**: экспоненциально взвешенное скользящее среднее
- **Isolation Forest**: метод изоляции леса
- **Тест Граббса**: статистический тест на выбросы

### Прогнозирование

- **Холт-Винтерс**: тройное экспоненциальное сглаживание с сезонностью

### Оценка эффективности

| Статус | Критерий |
|--------|----------|
| 🟢 Эффективный | < 90% от норматива |
| 🟡 Нормативный | 90-110% |
| 🟠 Предупреждение | 110-130% |
| 🔴 Критический | > 130% |
